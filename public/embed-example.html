Embed Examples

This page demonstrates the two ways to embed the Web Calendar widget.

1. With a pre-configured URL (no input field)

<div data-web-calendar data-url="https://example.com/feed.ics"></div>
<script src="https://your-server.com/web-calendar.js" data-server="https://your-server.com"></script>

2. Without a URL (shows input field)

<div data-web-calendar></div>
<script src="https://your-server.com/web-calendar.js" data-server="https://your-server.com"></script>
